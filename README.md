# Ajaxtype  
Pequeño proyecto para practicar peticiones AJAX y manipulación dinámica del DOM.

##  Funcionalidades
- Realiza una petición AJAX simulada.
- Muestra y actualiza información dinámicamente sin recargar la página.
- Manejo básico de eventos y actualización de interfaz.

##  Tecnologías
- HTML
- CSS
- JavaScript (AJAX / XMLHttpRequest o Fetch)

## Objetivo del proyecto
Este proyecto me ayudó a comprender:
- Cómo funcionan las peticiones asíncronas.
- Cómo actualizar contenido en tiempo real.
- Cómo estructurar un proyecto pequeño en JavaScript.

## Cómo ejecutarlo
Solo abre `index.html` en tu navegador.
